# Meu Negócio Hinode

Sistema completo de gestão desenvolvido para consultores e microfranqueados Hinode.

O sistema foi desenvolvido para controlar:

- Cadastro de Clientes
- Cadastro de Produtos
- Vendas
- Recebimentos
- Controle Financeiro
- Fluxo de Caixa
- Planejamento Financeiro
- Autenticação de Usuários
- Controle de Permissões
- Banco de Dados MySQL

---

# Tecnologias Utilizadas

## Backend

- Node.js
- Express
- Sequelize
- MySQL
- JWT
- bcrypt
- Helmet
- CORS

## Frontend

- HTML5
- CSS3
- JavaScript

# Instalação

Entre na pasta backend

```bash
cd backend
```

Instale as dependências

```bash
npm install
```

Inicie o servidor

```bash
npm start
```

Servidor:

```
http://localhost:3000
```
